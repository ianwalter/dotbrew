# dotbrew
> Ian's [Homebrew][brewUrl] configuration (part of [dotfiles][dotfilesUrl])

## Installation

Installs applications using [Homebrew][brewUrl], [Homebrew Cask][caskUrl],
[Homebrew Bundle][bundleUrl], and [mas-cli][masUrl]:

```console
./install.sh
```

## Updating

```console
./update.sh
```
